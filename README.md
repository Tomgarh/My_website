# Explore Nigeria

This is a website that showcases some of the most beautiful and interesting places in Nigeria.  
It features images, descriptions, and a responsive design for easy browsing.

## Features
- Explore cities like Lagos, Abuja, and Osun-Oshogbo
- Responsive layout for desktop and mobile
- Image gallery of Nigerian attractions
